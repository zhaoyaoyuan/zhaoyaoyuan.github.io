web核心面试题

  1. servlet什么时候创建

    1. 当浏览器第一次访问时有tomcat创建
    2. 可以通过配置文件

  2. servlet生命周期三个方法

    1. init
    2. service
    3. destory

  3. 如何解决请求和响应乱码

    1. request.setCharacterEncoding(“utf-8”)
    2. respones.setContextType(“text/html;character=utf-8”)

  4. session和cookie有什么区别

    1. 服务端 session 客户端 cookie

    2. 无大小限制 4kb 最多50个

    3. 相对安全 不安全

    4. session基于cookie

      ​ session创建成功后会以cookie形式将sessionid存入JSESSIONID

  5. jsp内置对象?

    1. 域对象
      1. pagecontext 可以获取其他8个域对象
      2. request
      3. session
      4. servletcontext
    2. out
    3. page
    4. config
    5. response
    6. exception

  6. el表达式四个域对象

    1. requestscope
    2. pageScore
    3. sessionscope
    4. ApplicationScope
