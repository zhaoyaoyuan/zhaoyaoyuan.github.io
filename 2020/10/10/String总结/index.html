String总结

  1. 创建

    1. String s = “abc”;
    2. String s = new String(“abc”);
    3. String s = new String(Char[] chars);
    4. String s = new String();

  2. 字符串比较

    1. String s = “abc”;
    2. s.equals(“abc”);返回boolean类型

  3. 字符串遍历

    1. String s = “abc”;
    2. S.toCharArray();返回一个字符数组
    3. for循环遍历字符数组
    4. 使用s.charAt()方法和s.length()方法获取字符串每个索引上的字符

  4. 统计字符次数

    1. 字符串遍历
    2. 使用if判断字符范围
    3. 计数器++

  5. 手机号屏蔽

    1. s.substring(索引);切割字符串
    2. 拼接字符串,中间+”****”+;

  6. 敏感词替换

    1. s.replace(目标字符/串,替换字符/串);
    2. 可以使用链式编程

  7. 切割字符串

    1. s.split(分隔符);按照分隔符切割字符串

    2. 得到字符串数组

    3. 遍历字符串数组进行操作
