二. 本地已经有代码 将代码推到 远程

  1. 先建立远程仓库 (码云, github )
  2. git remote add 远程别名 +地址
  3. git push -u 远程别名

4.写代码
5. git add .
6. git commit -m “”
7. git pull
8. git push
